This project proposal is for the development of a Plagiarism Checker website for ALX instructors as Final project assignment For ALX DevOps engineering certification. Plagiarism is a serious issue that needs to be controlled and monitored. It refers to the act of blindly copying someone else’s work and presenting it as your unique work. Here in Alx campus Plagiarism is done by students and affects the education quality of the students and thereby reduce the performance of students. .

About Us

Internet has changed the student’s life and also has changed their learning style. It allows the students to get deeper in the approach towards learning and making their task easier. Many methods are employed in detecting plagiarism. Usually plagiarism detection is done using text mining method. In this plagiarism checker website , After a student code sample uploaded from github link , then the new system further divides into content . This new web application will process the content, search google and github codebase to match the original content.

The development of a plagiarism checker for ALX instructors is a commendable initiative that addresses the critical issue of academic integrity. Plagiarism undermines the value of education and the authenticity of student work, so creating a robust system to detect and deter this practice is essential. By leveraging text mining techniques, the proposed system can analyze code samples uploaded via GitHub links, comparing them against a vast database of sources from Google and GitHub. This not only ensures the originality of student submissions but also upholds the standards of the ALX DevOps Engineering certification. Such a tool would be invaluable in maintaining the quality of education and fostering a culture of honesty and excellence among students.
